## Painter Usage
배경은 기본적으로 투명입니다.
지우개 사용시 clearRect로 똑같이 투명이 됨.

### Keyboard Hotkey
|Keyboard|Description|
|:-:|:-:|
|P|Pen|
|E|Eraser|
|G|Bucket|
|O|Pipette|
|[|decrease pen-linewidth|
|]|increase pen-linewidth|
